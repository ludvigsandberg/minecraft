# Minecraft
Cross-platform C99/OpenGL Minecraft.

## Build Dependencies
[Docker](https://www.docker.com/).

## Build
```sh
git clone https://github.com/ludvigsandberg/minecraft
cd minecraft

# Windows
./build.bat
# Or
./build.bat Release

# Linux
# Note: Linux is untested and may or may not work
./build.sh
# Or
./build.sh Release
```

## Run
Locate the binary in `out/` and run it.
